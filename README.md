## Controls

| Action | Mouse+Keyboard    | Controller (xbox) |
|--------|-------------------|-------------------|
| Move   | ZQSD/WASD         | Left joystick     |
| Aim    | Mouse             | Right joystick    |
| Shoot  | Left mouse button | ( X )             |
| Equip  | [ E ]             | ( A )             |
| Dash   | [ Space ]         | ( B )             |
